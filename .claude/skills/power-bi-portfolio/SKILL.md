## Generation Workflow

When user selects a project (e.g., "I choose HR Analytics"):

**Step 1: Create Todo List**
```
1. Generate Business Domain Education Guide for [Project]
2. Create [N] CSV dataset files with realistic data
3. Generate Data Dictionary and Data Model Diagram
4. Create 30-50 DAX formulas with explanations
5. Design Dashboard Blueprint with [N] pages
6. Write Presentation Script for demos
7. Create Portfolio Case Study document
8. Generate GitHub README and technical documentation
```

**Step 2: Execute Deliverables in Sequence**

For each deliverable:
- Mark todo as in_progress
- Generate complete, production-ready content
- Follow specifications exactly
- Ensure no placeholders or TODO items
- Mark todo as completed
- Move to next deliverable

**Step 3: Generate Supporting Assets**

- Create project folder: `[Number]_[Project_Name]/`
- Create subfolders: `documentation/`, `data/`, `scripts/`
- Generate Python script in `scripts/` folder
- Execute data generation script
- Verify all CSV files created in `data/` folder
- Place README.md in project root

**Step 4: Final Summary**

Provide user with:
- Files created count and locations
- Total documentation pages
- Total data rows generated
- Next steps for user
- Build timeline estimate
- Contract value potential
